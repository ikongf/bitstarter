Bitstarter Title

This is my first bitstarter project.

I am interested in creating a data analysis tool.

The disruptive and growth potential of a tool is due to a few trends: so called big data, lack of expertise in data science skills including statistics, machine learning, and programming languages such as python, Pig, Hive, etc. Further, the high end analysis is not the key need and the market is over-served. The larger opportunity is helping ordinary business people and analysts, mostly doing data manipulation, data wrangling, data blending, etc. and some basic reporting. These folks need GUI tools to answer questions. They do not want to learn Hive, Pig, python, Java, Map Reduce, etc. The technology change is incredibly rapid. Hadoop Map Reduce and so on is quickly being outmoded by efforts such as Drill, Impala, Spark, and Shark.

So we will create a basic tool that allows business users to answer questions with very fast speed due to the underlying new engines, likely using the Amazon web services. This tool can address "Medium Data", between spreadsheets and real Big Data that mainly Internet companies need to analyze for ad serving and recommendations.
